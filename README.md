# dp-kalkulator-frontend

Standalone frontend for dagpengekalkulator.

## Stack

- React Router (framework mode)
- React 19
- RVF dependency (`@rvf/react-router`)
- NAV Aksel styles

## Run locally

1. Install Node from `.nvmrc`
2. Install dependencies:
   - `pnpm install`
3. Start dev server:
   - `pnpm dev`

## Notes

- All calculator text is hardcoded in the app.
- No Sanity dependency is used.
- Calculation and date logic is copied from `dp-produktside-frontend` calculator utilities.
