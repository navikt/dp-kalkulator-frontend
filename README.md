## Utvikling

### Utvikle lokalt

`npm install`

`npm start`

For å mocke api'et kan du gå i filen `src/Api/index.js` og sette `const mock = true;`

### Lokal server

Installer server-deps:

`npm --prefix ./server install`

`npm run server`

### Bygg og kjør docker-container

`npm run docker`
